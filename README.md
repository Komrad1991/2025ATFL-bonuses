# Ограничения на автомат:
1)нет эпсилон переходов\
2)определены все переходы\
3)начальное состояние с типом bald (см. пример)\
4)финальные состояние: круг в круге\

# Ограничения на сеть петри:
1)места - кружки, переходы - квадраты, дуги - направленные\
2)название мест - p[i], название переходов - t[i]\
3)начальная маркировка содержит все места

# Ограничения на очистку грамматики (только от непродуктивных и недостижимых)
1)Терминалы: маленькие символы\
2)Нетерменалы: большие\
3)входной файл - txt с описанием грамматики, построчно. в одной строчке разделение символом '|'
